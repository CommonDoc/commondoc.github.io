Codex

Build beautiful documentation

Codex is a modern documentation generator for Common Lisp. It makes it easy to create beautiful and useful documentation from both prose and automatically extracted docstrings.

Codex uses CommonDoc due to its expressive power and macro system.

The source is available here. Start at the tutorial.
